## 🚗 What is TaxiAgent?

TaxiAgent is a desktop-ready project for ride-hailing work flows. It uses a multi-agent design with Spring Boot and Spring AI to help manage tasks that fit a taxi or ride-hailing scene.

This project is built from the ground up around a ReAct-style framework. That means it can think through a task, take an action, and then decide the next step. For an end user, the goal is simple: run the app on Windows and use it to handle taxi-related work in a clear way.

## 🧩 How TaxiAgent Works

TaxiAgent uses a multi-agent structure. In plain terms, that means different parts of the system can focus on different jobs.

A common flow looks like this:

- one agent reads the task
- one agent decides what to do next
- one agent calls the right tool or service
- one agent checks the result
- the system repeats the loop until the task is done

This ReAct-style design helps the app handle tasks that need step-by-step thinking.
